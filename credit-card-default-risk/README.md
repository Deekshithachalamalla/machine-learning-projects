# Credit Card Default Risk Analysis

## Problem Statement
Predict whether a credit card customer will default based on demographic,
financial, and repayment history data.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
## Dataset
The dataset used in this project is stored in the main `datasets/` directory of this repository.
